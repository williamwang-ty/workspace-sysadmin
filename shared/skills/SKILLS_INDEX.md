# 技能索引路由表 (Skills Index)

本文件供无原生扫描能力的 Agent（如 Goose）进行二次路由查找合适的技能 SOP。

- **mac-system-diagnostics**: 全面的 macOS 系统健康检查、硬件与网络排错。请参阅 `shared/skills/mac-system-diagnostics/SKILL.md`。
- **cleanup-and-uninstall**: 深度清理与卸载审计。请参阅 `shared/skills/cleanup-and-uninstall/SKILL.md`。
- **homebrew-expert**: Homebrew 软件包管理、更新与备份。请参阅 `shared/skills/homebrew-expert/SKILL.md`。
- **fix-issue**: 系统报错、崩溃或其他异常的自动定位与修复。请参阅 `shared/skills/fix-issue/SKILL.md`。

遇到相关任务时，请主动根据此表寻找对应的 SOP 文件进行阅读。
